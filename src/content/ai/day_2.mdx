## 常量
用`final`来定义一个常量

## static
属于类，生命周期和类一样
static方法、属性可以被该类对象实例共享使用
static方法不能调非static方法
### 静态代码块用于类的初始化
没啥屌用
```java
Class Car {
  static int speed;

  static {
    speed = 50
  }
}
```

## 简单内存模型

变量需要内存中申请空间
申请出的空间只能用于存储该类型的数据

基本类型：存储再stack，直接存储
引用类型：存储再heap，stak区存储的是对真实对象的地址引用

### stack
先进后出 LIFO，会创建栈针出入栈。栈针会存局部变量表等、常量、引用对象的地址

### heap
无序
保存new出来的成员对象变量，保存动态数据

比如
```java
user 
  id
  name
  sayHello
```

string、sayHello就保存一个地址，指向方法区

### 方法区
线程
可以理解成被共享的区域，保存类信息
大小决定了能保存多少类（字段、方法、常量池），多了就溢出了

## this
- 构造器中this指向正要初始化的对象地址
- 普通方法中this指向调用该方法的对象地址
- this不能用在出现static的地方
- this()：调用构造方法，必须在第一行
