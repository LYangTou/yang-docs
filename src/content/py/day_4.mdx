## def

### 返回多个值：
```python
def return_multiple_values():
    return 1, 2, 3

a, b, c = return_multiple_values()
print(a, b, c)  # 1 2 3
```

但是实际上返回的是一个元组，所以可以这样：
```python
def return_multiple_values():
    return 1, 2, 3

a = return_multiple_values()
print(a)  # (1, 2, 3)
```

### 默认参数：
```python
def default_parameter(a, b=10):
    return a + b

print(default_parameter(1))  # 11
print(default_parameter(1, 2))  # 3
```

### 带参数名的参数：
```python
def named_parameter(a, b=10):
    return a + b

print(named_parameter(b=1, a=2))  # 3
```
带参数名的参数可以不按照顺序传递参数，但是需要指定参数名。

### 可变参数：
```python
def variable_parameter(*args):
    return args

print(variable_parameter(1, 2, 3))  # (1, 2, 3)
```

可变参数可以接收任意数量的参数，参数会以元组的形式传递给函数。

```python
nums = [1, 2, 3]
calc(*nums)
14
```
在list或tuple前面加一个*号，把list或tuple的元素变成可变参数传进去

### 关键字参数：
```python
def keyword_parameter(**kwargs):
    return kwargs
```

关键字参数可以接收任意数量的参数，参数会以字典的形式传递给函数。

### 分隔符
```python
def person(name, age, *, city, job):
    print(name, age, city, job)
```
分隔符后面的参数必须传参数名，否则会报错。
```python
person('John', 20, 'Beijing', 'Engineer')  # 报错
person('John', 20, city='Beijing', job='Engineer')  # 不报错
```
如果函数定义中已经有了一个可变参数，后面跟着的命名关键字参数就不再需要一个特殊分隔符*了
