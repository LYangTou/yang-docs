# HashMap
Hash table based implementation of the Map interface.
HashMap就是基于哈希表的Map接口实现。

## 哈希表

什么是哈希表？
- 设计精妙，用途广泛的数据结构之一
- 拥有键值对元素的无序集合
- 键的值是唯一的，键对应ing的值可以通过键来获取/更新/一处
- 无论这个哈希表多大，这些操作（Insert/Search/Delete）基本上通过常量时间O(1)的键比较就可以完成。

### 为什么设计哈希表？

数据结构可以分为两大类
- 连续型（比如array），在内存中是连续的，访问很快，但是删除或中间插入就比较慢。
- 离散型（单链表）,查询慢，增加删除快

哈希表就是兼顾两种的优点。

## HashMap使用
```java
    Map<String, Double> map = new HashMap<>();

    // size/isEmpty
    System.out.println(map.size());
    System.out.println(map.isEmpty());

    // put/putALL/putIfAbsent
    // 1. put 把键值对放入hashmap (包含增和改)
    map.put("abc", 2.3);
    map.put("ab0", 2.56);
    // 覆盖原来的键值对，除非明确知道要更新键值对的时候用重复key调用put方法
    // 返回值会返回之前的key的value，如果没有返回null，可以根据这个判断是增加还是更新
    map.put("abc", 2.5);

    Double value = map.put("c", null);
    System.out.println(value);
    value = map.put("c", null);
    System.out.println(value);
    // 2. putIfAbsent
    // 如果原来的key有值，则直接返回原来的值不更新Key，原来没有返回null并更新key
    value = map.putIfAbsent("abc", 5.5);
    System.out.println(map.get("abc"));

    // 3. putAll 一般不建议用
    Map<String, Double> newMap = new HashMap<>();
    // 有相同的键会覆盖之前的
    newMap.put("abc", 10.2);
    newMap.put("dd", 10.2);
    newMap.put("cd", 10.2);
    map.putAll(newMap);
    System.out.println(map.get("abc"));

    // get/getOrDefault

    // 1. get方法：对应key有值返回值，没有返回null
    map.get("dd");
    // 2. getOrDefault
    Double value1 = map.getOrDefault("ddd", .2);
    System.out.println(value1);
    System.out.println(map.get("ddd"));

    // containsKey/containsValue
    System.out.println(map.containsKey("ddd"));
    System.out.println(map.containsValue(10.2));

    // remove 有值返回值，没值返回null
    Double r = map.remove("dd");
    System.out.println(r);
    // 需要key 和 value都对应才能删除
    boolean s = map.remove("abc", 10.2);
    System.out.println(s);

    // replace 代替put做更新，如果key不存在直接返回null
    map.replace("cd", 123.5);
    System.out.println(map.get("cd"));

    // keySet/values/entrySet
    Set<String> keys = map.keySet();

    Collection<Double> values = map.values();

    Set<Map.Entry<String, Double>> entrys = map.entrySet();
    for (Map.Entry<String, Double> entry : entrys) {
        String key = entry.getKey();
        Double v = entry.getValue();
        System.out.println(key+ ":" + v);
    }

    // 遍历map
    // 1. 遍历entry
    for (Map.Entry<String, Double> entry : map.entrySet()) {
        System.out.println(entry);
    }
    // 2. 通过keySet遍历key
    for (String key: map.keySet()) {
        System.out.println(map.get(key));
    }
    // 3.利用set的迭代器 一般用不上
    // 1) entrySet迭代器
    Iterator<Map.Entry<String, Double>> it = map.entrySet().iterator();
    while (it.hasNext()) {
        Map.Entry<String, Double> entry = it.next();
        System.out.println(entry);
    }
    // 2)利用keySet迭代器
    Iterator<String> it2 = map.keySet().iterator();
    while (it2.hasNext()) {
        String key = it2.next();
        System.out.println(map.get(key));
    }
    // 4. lambda
    map.forEach((k, v) -> System.out.println(k+ ':' + v));
```
