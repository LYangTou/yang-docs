## 哈希表基本原理
- 数组通过下标访问数据的一种拓展
- 核心：利用哈希函数将键值映射到数组上
  比如"abc"根据ascii码转成数字取模获得数组下标。当不同的key计算出的数字相同时也就是哈希冲突

### 哈希函数
- 哈希函数式用来将一个字符串（或任何其他类型）转化为小于哈希表大小且大于等于0的整数
一个好的哈希函数
1. 可以尽可能少的产生哈希冲突
2. 算的快

一种广泛使用的哈希函数算法是使用数值33（Times33算法）,java里面是改成了31
```java
public int hashCode(char[] key, int hashSize) {
    long result = 0;
    for (int i = 0;i < key.length;i++) {
        result = (result * 33 + (int)(key[i])) % hashSize
    }
    return (int)result
}
```

### 哈希冲突
- 表面原因：有一些key会map到相同index上
- 本质：无限空间往有限空间映射

如何解决哈希冲突
- 设计好的hash函数
- 改变哈希索引
  - Open hashing
  - Closed hashing
- 扩容

### Closed hashing 闭散列
开放定址法
线性探测
``` 
hash = (hash(key) + i) % hashSize
i = 0,1,2,3....
```
也就是取模探测到相同key值的时候加一个i，直到整个哈希表被填满
设置和查找是同样的操作。当查找的时候遍历到空值时，就说明要查找的key不在散列表中。
删除的时候要做一个delete标记，这样查找的时候不影响。

问题是空间小，并且遍历时间复杂度高。删除等操作复杂。

无论是线性探测/二次探测/双重散列都有空间问题，导致潜在冲突增加。

### Open Hash 开散列

- 每个bucket对应一条链表，哈希值相同的元素直接连接在对应链表中
- 链表的头结点存储在哈希表中

缺点：复杂、不能直接访问元素，要间接访问指针。

极端情况所有key都在一个bucket中，链表非常长，导致变成了单纯的链表查找。

java 7底层数据结构式数组+链表  8 是数组+链表+红黑树。

### 扩容：重哈希rehashing
- 哈希表容量的大小在一开始是不确定的，在需要的时候，可以对底层数组进行扩容
- 一种简单策略：如果哈希表存储元素太多，将哈希表容量扩大一倍，并将所有的key的哈希值重新计算映射到新的bucket上。

```java
public class MyHashMap<K,V> {
    // JAVA底层默认数组长度都是2的整数次幂
    private static final int DEFAULT_INIT_CAPACITY = 16;

    private static final float DEFAULT_INIT_FACTORY = 0.75f;
    // Fields
    static class Node<K,V> {
        K key;
        V value;
        Node<K, V> next;

        public Node(K key, V value, Node<K, V> next) {
            this.key = key;
            this.value = value;
            this.next = next;
        }

        public K getKey() {
            return key;
        }

        public void setKey(K key) {
            this.key = key;
        }

        public V getValue() {
            return value;
        }

        public void setValue(V value) {
            this.value = value;
        }

        public Node<K, V> getNext() {
            return next;
        }

        public void setNext(Node<K, V> next) {
            this.next = next;
        }

        @Override
        public String toString() {
            return "Node{" +
                    "key=" + key +
                    ", value=" + value +
                    ", next=" + next +
                    '}';
        }
    }
    private int intCapacity;

    // 装填因子LF
    private float loadFactory;

    private int size;

    // 采用数组+链表作为底层存储结构

    private Node<K, V>[] table;

    // Constructors

    public MyHashMap() {
        this(DEFAULT_INIT_CAPACITY, DEFAULT_INIT_FACTORY);
    }

    public MyHashMap(int intCapacity, float loadFactory) {
        this.intCapacity = intCapacity;
        this.loadFactory = loadFactory;
        this.table = new Node[intCapacity];
    }

    // Methods

    public int size() {
        return this.size;
    }

    public V put(K key, V value) {
        if (size() >= intCapacity * loadFactory) {
            resize(intCapacity * 2);
        }
        int index = hash(key);
        Node<K, V> node = table[index];
        if (node == null) {
            // 哈希桶为空，插入新节点
            this.table[index] = new Node<>(key ,value, null);
            size++;
            return null;
        }
        // 哈希桶不为空 两种情况：

        while (node != null) {
            // 1. key存在，更新value。
            if (node.key == key) {
                V oldValue = node.getValue();
                node.setValue(value);
                return oldValue;
            }
            // 2. key不存在 新增
            if (node.getNext() == null) {
                // key 不存在。新增key-value
                // 尾插法
                node.setNext(new Node<>(key ,value, null));
                size++;
            }
            node = node.getNext();
        }
        return null;
    }

    private void resize(int newCapacity) {
        System.out.println("resize" + newCapacity);
        Node<K, V>[] newTable = new Node[newCapacity];
        this.intCapacity = newCapacity;
        // rehashing: 遍历老的哈希表

        for (Node<K, V> node: table) {
            // 遍历链表
            while (node != null) {
                // 在新hash表的位置
                int index = hash(node.key);
                if (newTable[index] != null) {
                    // 找到合适位置
                    Node<K, V> head = newTable[index];
                    while (head.getNext() != null) {
                        head = head.getNext();
                    }
                    Node<K, V> newNode = new Node<>(node.getKey(), node.getValue(), null);
                    head.setNext(newNode);
                } else {
                    // 直接插入到头节点
                    Node<K, V> newNode = new Node<>(node.getKey(), node.getValue(), null);
                    newTable[index] = newNode;
                }
                node = node.getNext();
            }
        }

        this.table = newTable;
    }

    private int hash(K key) {
        return Math.max(key.hashCode() % intCapacity, 0);
    }

    public V get(K key) {
        // 确定哈希桶
        int index = hash(key);

        Node<K, V> node = table[index];

        while (node != null) {
            if (node.getKey() == key) {
                return node.getValue();
            }
            node = node.getNext();
        }

        return null;
    }

    public V remove(K key) {
        int index = hash(key);
        Node<K, V> node = table[index];
        // 针对于头节点逻辑
        if (node == null) {
            return null;
        }

        if (node.getKey() == key) {
            // 保存被删除掉的next
            Node<K, V> temp = node.getNext();
            V value = node.getValue();
            node.setNext(null);
            table[index] = temp;
            size--;
            return value;
        }

        Node<K, V> prevNode = node;
        Node<K, V> curNode = node.next;
        while (curNode != null) {
            // 找到对应的k v
            if (curNode.getKey() == key) {
                prevNode.setNext(curNode.getNext());
                curNode.setNext(null);
                size--;
                return curNode.getValue();
            }

            prevNode = curNode;
            curNode = curNode.getNext();
        }
        return null;
    }

    public static void main(String[] args) {
        MyHashMap<Integer, String> hashMap = new MyHashMap<>();
        hashMap.put(10, "a");
        hashMap.put(22, "b");
        hashMap.put(6, "c");
        hashMap.put(54, "d");
        System.out.println(hashMap.size());
        for (Node<Integer, String> node : hashMap.table) {
            System.out.println(node);
        }

        System.out.println(hashMap.get(10));
        System.out.println(hashMap.get(22));
        System.out.println(hashMap.get(6));
        System.out.println(hashMap.get(54));
        System.out.println("===================");
        System.out.println(hashMap.remove(6));
        System.out.println(hashMap.get(6));
        System.out.println(hashMap.remove(10));
        System.out.println(hashMap.get(10));
        System.out.println(hashMap.remove(166));
        for (Node<Integer, String> node : hashMap.table) {
            System.out.println(node);
        }
        System.out.println(hashMap.size());

        MyHashMap<String, String> stringMap = new MyHashMap<>(5, 0.5f);
        MyHashMap<Integer, Integer> intHm = new MyHashMap<>(5, 0.75f);
        intHm.put(10, 3);
        intHm.put(3, 3);
        intHm.put(5, 3);
        intHm.put(4, 3);
        intHm.put(9, 3);
        intHm.put(2, 3);
        intHm.put(8, 3);
        intHm.put(7, 3);
        intHm.put(11, 3);
        intHm.put(12, 3);

        for (Node<Integer, Integer> node : intHm.table) {
            System.out.println(node);
        }
    }

}
```
