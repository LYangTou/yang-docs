### Java程序是如何编译和运行的
1. 编译
    Main.java -> javac -> Main.class
    1. 写源代码 Main.java
    2. 编译 Main.class 字节码文件
2. 运行 在虚拟机上运行（JVM）,通过类加载器 加载Main.class 运行

### 变量的分类和作用域

#### 变量分类
- Class Variables (Static Fileds) 类变量
- Instance Variables (Non-static Fileds) 实例变量
- Local Variables 局部变量
- Parameters方法参数

## 内部类
- 目的：更加细力度的封装
- 非静态内部类可以使用外部类的成员方法、属性 编译后的字节码内部类会绑一个 this$0指向外部类
- 静态内部类无法直接使用外部类的成员方法、属性 编译后的字节码内部类没有这个this$0，可以通过那构造器注入的方式调
- 分类：非静态内部类、静态内部类、匿名内部类

## 匿名内部类
- 没有名字的类，内部类的简化写法
- 本职：继承该类或者实现接口的子类匿名对象
