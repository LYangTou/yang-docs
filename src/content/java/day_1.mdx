# Java 基础类型、包装类、String、数组 笔记

## 一、基础类型

### 1. byte

* 数值范围：`-128 ~ 127`
* 占用空间：**1字节**
* 最高位是符号位：`0` 表示正数，`1` 表示负数
* 负数以**补码**形式存储
* 应用场景：文件读写、网络传输中的字节数据

---

### 2. short

* 数值范围：`-32768 ~ 32767`
* 占用空间：**2字节**
* 注意：`byte`、`short`、`char` 参与算术运算时，都会先提升为 `int`

```java
short a = 1;
short b = 2;
// short c = a + b; // 报错
int c = a + b;
```

---

### 3. int

* 数值范围：`-2^31 ~ 2^31 - 1`
* 占用空间：**4字节**
* 整数字面量默认就是 `int`

```java
int a = 100;
```

---

### 4. long

* 数值范围：`-2^63 ~ 2^63 - 1`
* 占用空间：**8字节**
* 如果数值超出 `int` 范围，必须加 `L`

```java
long big = 3000000000L;
```

原因：

* 整数字面量默认是 `int`
* `3000000000` 已经超出 `int` 范围，所以必须加 `L`

---

### 5. float

* 单精度浮点数
* 占用空间：**4字节**
* 必须加 `F` 或 `f`

```java
float f = 1.23F;
```

原因：

* 小数字面量默认是 `double`
* `double` 不能直接赋值给 `float`

---

### 6. double

* 双精度浮点数
* 占用空间：**8字节**
* 小数默认类型就是 `double`

```java
double d = 1.23;
```

---

### 7. 浮点数精度问题

* `float` / `double` 适合近似计算
* 不适合精确比较和金额计算

```java
System.out.println(0.1 + 0.2 == 0.3); // false
```

原因：

* 二进制无法精确表示某些十进制小数

精确计算要用 `BigDecimal`

```java
BigDecimal a = new BigDecimal("0.1");
BigDecimal b = new BigDecimal("0.2");
System.out.println(a.add(b).equals(new BigDecimal("0.3"))); // true
```

---

### 8. char

* 占用空间：**2字节**
* 表示一个字符，本质上保存的是一个 **UTF-16 编码单元**
* 范围：`\u0000 ~ \uFFFF`

```java
char c = 'A';
char c2 = '中';
```

补充：

* `char` 可以参与运算，运算时会转成整数

```java
char c = 'A';
int num = c;   // 65

int x = 66;
char c2 = (char) x; // 'B'
```

---

### 9. boolean

* 只有两个值：`true` 和 `false`
* Java 语言层面不规定它具体占多少字节
* 学习基础时只需要记住它表示真假即可

---

## 二、包装类

### 1. 对应关系

| 基础类型    | 包装类       |
| ------- | --------- |
| byte    | Byte      |
| short   | Short     |
| int     | Integer   |
| long    | Long      |
| float   | Float     |
| double  | Double    |
| char    | Character |
| boolean | Boolean   |

---

### 2. 推荐写法

不推荐记 `new Integer(10)` 这种旧写法，推荐：

```java
Integer a = Integer.valueOf(10);
Integer b = 10; // 自动装箱
```

---

### 3. 自动装箱与拆箱

```java
Integer num = 10; // 自动装箱
int x = num;      // 自动拆箱
```

本质：

* 装箱：`Integer.valueOf(10)`
* 拆箱：`num.intValue()`

---

### 4. 自动拆箱空指针问题

```java
Integer a = null;
// int b = a; // NullPointerException
```

原因：

* 自动拆箱本质上会调用 `a.intValue()`
* `a` 为 `null` 时会空指针

---

### 5. 缓存池

`Integer` 等包装类对部分范围的值有缓存机制：

```java
Integer a = 100;
Integer b = 100;
System.out.println(a == b); // true

Integer c = 200;
Integer d = 200;
System.out.println(c == d); // false
```

结论：

* `==` 比较的是地址
* `equals()` 比较的是值
* 比较包装类内容时，优先用 `equals()`

```java
System.out.println(c.equals(d)); // true
```

---

## 三、String

### 1. String 的特点

* `String` 是**不可变对象**
* 字符串内容一旦创建，就不能修改

```java
String s = "abc";
s += "d";
```

这里不是修改原字符串，而是创建了一个新字符串。

---

### 2. 字符串常量池

直接写双引号的字符串会进入常量池：

```java
String a = "abc";
String b = "abc";
System.out.println(a == b); // true
```

`new String("abc")` 创建的是堆中的新对象：

```java
String a = "abc";
String b = new String("abc");

System.out.println(a == b);      // false
System.out.println(a.equals(b)); // true
```

---

### 3. 常用方法

#### 获取长度

```java
String s = "hello";
System.out.println(s.length());
```

#### 获取指定位置字符

```java
char c = s.charAt(1);
```

#### 截取字符串

```java
String sub = s.substring(1, 4); // 左闭右开
```

#### 拼接

```java
String result = s1.concat(s2);
```

#### 替换

```java
String replaced = s.replace("a", "A");
```

#### 分割

```java
String ip = "192.168.1.1";
String[] arr = ip.split("\\.");
```

注意：

* `split()` 传入的是**正则表达式**
* `"."` 在正则里有特殊含义，所以要写 `\\.`

---

### 4. 判空写法

```java
if (str != null && !str.isEmpty()) {
}
```

区别：

* `null`：对象不存在
* `isEmpty()`：长度是否为 0

如果想判断去掉空格后是否为空：

```java
if (str != null && !str.trim().isEmpty()) {
}
```

## 四、BigDecimal

### 1. 推荐写法

```java
BigDecimal a = new BigDecimal("0.1");
```

不推荐：

```java
new BigDecimal(0.1)
```

因为 `0.1` 本身作为 `double` 就不精确。

---

### 2. 常用操作

```java
BigDecimal a = new BigDecimal("10.5");
BigDecimal b = new BigDecimal("2");

a.add(b);        // 加
a.subtract(b);   // 减
a.multiply(b);   // 乘
a.divide(b);     // 除
```

---

### 3. 比较大小

比较 `BigDecimal` 不建议直接用 `equals()` 判断数值大小，通常用 `compareTo()`

```java
BigDecimal a = new BigDecimal("2.0");
BigDecimal b = new BigDecimal("2.00");

System.out.println(a.equals(b));      // false
System.out.println(a.compareTo(b));   // 0
```

## 五、数组

### 1. 数组的概念

数组是：**存放同一种类型多个数据的容器**

特点：

* 元素类型相同
* 长度固定
* 可以通过下标访问
* 下标从 `0` 开始

---

### 2. 数组定义

#### 静态初始化

```java
int[] arr = {1, 2, 3};
```

#### 动态初始化

```java
int[] arr = new int[3];
```

---

### 3. 数组默认值

#### 基本类型数组

* `int`：`0`
* `double`：`0.0`
* `char`：`\u0000`
* `boolean`：`false`

#### 引用类型数组

* `null`

```java
int[] arr1 = new int[3];
String[] arr2 = new String[3];
```

---

### 4. 数组访问

```java
int[] arr = {10, 20, 30};

System.out.println(arr[0]); // 10
arr[1] = 99;
System.out.println(arr[1]); // 99
```

---

### 5. 数组长度

```java
System.out.println(arr.length);
```

注意：

* 数组：`length`
* 字符串：`length()`

---

### 6. 数组遍历

#### 普通 for

```java
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

#### 增强 for

```java
for (int num : arr) {
    System.out.println(num);
}
```

区别：

* 普通 `for`：可以拿到下标
* 增强 `for`：适合只遍历

---

### 7. 二维数组

#### 定义

```java
int[][] arr = {
    {1, 2, 3},
    {4, 5, 6}
};
```

#### 访问

```java
System.out.println(arr[0][1]); // 2
```

#### 遍历

```java
for (int i = 0; i < arr.length; i++) {
    for (int j = 0; j < arr[i].length; j++) {
        System.out.print(arr[i][j] + " ");
    }
    System.out.println();
}
```

---

### 8. 数组常见异常

#### 下标越界

```java
int[] arr = {1, 2, 3};
// System.out.println(arr[3]); // ArrayIndexOutOfBoundsException
```

#### 空指针

```java
int[] arr = null;
// System.out.println(arr.length); // NullPointerException
```

---

## 六、Arrays 常用方法

```java
import java.util.Arrays;
```

### 1. 打印数组

```java
int[] arr = {1, 2, 3};
System.out.println(Arrays.toString(arr));
```

二维数组：

```java
int[][] arr2 = {{1, 2}, {3, 4}};
System.out.println(Arrays.deepToString(arr2));
```

---

### 2. 排序

```java
int[] arr = {5, 2, 8, 1};
Arrays.sort(arr);
System.out.println(Arrays.toString(arr)); // [1, 2, 5, 8]
```

---

### 3. 比较数组内容

```java
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};

System.out.println(Arrays.equals(a, b)); // true
```

注意：

* `==` 比地址
* `Arrays.equals()` 比内容

---

### 4. 复制数组

#### copyOf

```java
int[] arr = {1, 2, 3};
int[] newArr = Arrays.copyOf(arr, 5);
System.out.println(Arrays.toString(newArr)); // [1, 2, 3, 0, 0]
```

#### copyOfRange

```java
int[] arr = {1, 2, 3, 4, 5};
int[] sub = Arrays.copyOfRange(arr, 1, 4);
System.out.println(Arrays.toString(sub)); // [2, 3, 4]
```

---

### 5. 填充数组

```java
int[] arr = new int[5];
Arrays.fill(arr, 7);
System.out.println(Arrays.toString(arr)); // [7, 7, 7, 7, 7]
```

---

### 6. 二分查找

前提：数组必须有序

```java
int[] arr = {1, 3, 5, 7, 9};
int index = Arrays.binarySearch(arr, 5);
System.out.println(index); // 2
```

没找到时返回负数。

## 七、数组常见基础操作

### 1. 求最大值

```java
int[] arr = {3, 9, 2, 7};

int max = arr[0];
for (int i = 1; i < arr.length; i++) {
    if (arr[i] > max) {
        max = arr[i];
    }
}
System.out.println(max);
```

---

### 2. 求最小值

```java
int min = arr[0];
for (int i = 1; i < arr.length; i++) {
    if (arr[i] < min) {
        min = arr[i];
    }
}
System.out.println(min);
```

---

### 3. 求和

```java
int sum = 0;
for (int num : arr) {
    sum += num;
}
System.out.println(sum);
```

---

### 4. 查找某个元素

```java
int target = 7;
int index = -1;

for (int i = 0; i < arr.length; i++) {
    if (arr[i] == target) {
        index = i;
        break;
    }
}
System.out.println(index);
```

---

### 5. 反转数组

```java
int[] arr = {1, 2, 3, 4, 5};

for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
```

## 八、这一部分最该记住的点

### 基础类型

* `byte`、`short`、`char` 运算时先提升为 `int`
* 整数字面量默认是 `int`
* 小数字面量默认是 `double`
* 金额计算不用 `float/double`，用 `BigDecimal`

### 包装类

* 自动拆箱可能空指针
* 比较包装类内容用 `equals()`

### String

* `String` 不可变
* `==` 比地址，`equals()` 比内容
* `split()` 参数是正则

### 数组

* 数组长度固定
* 下标从 `0` 开始
* 常用工具类是 `Arrays`
