# ArrayList
- 基于数组实现的容量大小可动态变化的数据结构
- 可以将很多数组操作的细节封装起来
- 特点：查询效率高，增加删除效率低
- ArrayList是List接口的可变数组实现，实现了List接口
- ArrayList是线程不安全的
- Vector类是上古时代的ArrayList，现在不用了


- size 实际长度
- capacity 数组容量
